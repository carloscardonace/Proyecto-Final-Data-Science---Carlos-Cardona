# 🛡️ Proyecto de Detección de Fraude en Transacciones Financieras

## 🎯 Objetivo del Proyecto

Desarrollar un modelo de clasificación binaria que permita predecir si una transacción financiera es fraudulenta o no, utilizando un enfoque basado en regresión logística. El modelo tiene como propósito reforzar los conocimientos adquiridos en el curso de Data Science, incluyendo limpieza de datos, análisis exploratorio, ingeniería de características, selección de variables, evaluación de modelos y documentación técnica.
